Info about the app:

The app is Windows Desktop App.
You will be needing Microsoft Windows Visual Studio 2015 to run this app.
The language used for coding is C# and for storing data .txt files have been used.
The app is fully offline and no nett is required, app size is less than 5Mb.
For demoability multiple users can login in the same instance of the app.
The data in the app is consistent and persists even after the app is closed.
You will get few message dialog box, click on "Ok" to move forward.

To run the app:

Download the whole project.
Go to: The folder containing Microsoft Visual Studio (usually Documents) -> Microsoft Visual Studio -> Projects -> Paste the downloaded project there.
After pasting it in right location open the project and then double click on WindowsFormsApplication.sln file and then run the app by clicking at the start button at top mid of the visual studio.
